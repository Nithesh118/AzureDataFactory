# Azure Data Engineering Pipeline (ADF Project)

## 🚀 Overview
This project demonstrates an end-to-end data pipeline built using Azure Data Factory with Medallion Architecture.

## 🏗 Architecture
- Data Sources: ADLS, GitHub, API
- Ingestion: Azure Data Factory
- Storage: ADLS (Bronze, Silver, Gold)
- Transformation: ADF Data Flows
- Orchestration: Parent Pipeline

## 🔄 Pipeline Flow
1. Data ingestion from multiple sources
2. Raw data stored in Bronze layer
3. Data cleaning in Silver layer
4. Aggregated data in Gold layer

## ⚡ Features
- Incremental loading using watermark
- Modular pipelines
- End-to-end orchestration

## 🛠 Tech Stack
- Azure Data Factory
- Azure Data Lake Storage
- GitHub
